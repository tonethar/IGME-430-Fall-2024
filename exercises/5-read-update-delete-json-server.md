# Read/Update/Delete with `json-server`

## II. Overview
- Do you know what CRUD is?
  - **C**reate
  - **R**ead
  - **U**pdate
  - **D**elete
- HTTP Verbs 
  - **C**reate - POST (create a resource)
  - **R**ead - GET
  - **U**pdate - PUT (usually used to completely replace resource), PATCH (partially update a resource)
  - **D**elete - DELETE
